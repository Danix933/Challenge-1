# Análisis de Tiendas

## Descripción

Este proyecto consiste en un análisis de datos de cuatro tiendas con el objetivo de comparar su desempeño y tomar una decisión estratégica sobre cuál debería vender el Sr. Juan.

Se analizaron diferentes factores como ingresos, ventas por categoría, calificaciones de clientes y costos de envío.

---

## Objetivo

Comparar las tiendas utilizando datos reales para identificar cuál tiene menor rentabilidad y justificar su posible venta.

---

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

---

## Qué se analizó

- Ingresos totales por tienda  
- Categorías más y menos vendidas  
- Calificación promedio de los clientes  
- Costo de envío promedio  

También se generaron gráficos de barras y gráficos circulares para visualizar mejor los resultados.

---

## Conclusión

Después del análisis, se determinó cuál tienda presenta menor desempeño general, considerando ingresos, ventas y satisfacción del cliente.

La recomendación final se encuentra explicada dentro del notebook del proyecto.

---

3. Revisar los gráficos y conclusiones finales.
